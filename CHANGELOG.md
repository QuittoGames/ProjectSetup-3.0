# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-27

### Added
- Gerador automático de estruturas de projetos
- Suporte para 40+ linguagens de programação
- Interface Rich com tema moderno azul e preto
- Templates personalizáveis via arquivos JSON
- Executável standalone para Windows (.exe)
- Publicação no PyPI
- Comandos CLI: `ps3` e `ps3cli`
- Pastas de fontes customizadas
- Sistema de ícones com Nerd Fonts
- Serviços modulares (ArrowsService, CLIService, ProjectManagerService)
- Configuração flexível via Config.py
- Suporte para histórico de projetos
- Dashboard principal interativo

### Features
- **40+ Linguagens Suportadas**: Python, JavaScript, TypeScript, Java, C++, Go, Rust, e muito mais
- **Interface Rica**: Terminal com interface bonita usando Rich library
- **Personalização Total**: Edite templates JSON para criar suas próprias estruturas
- **Multiplataforma**: Funciona em Windows, Linux e macOS
- **Zero Configuração**: Funciona out-of-the-box após instalação

### Technical
- Python 3.10+ required
- Dependencies: rich>=13.0.0
- Build system: setuptools
- Packaging: PyInstaller for executables
